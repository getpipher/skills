# @getpipher/workspace

repo catch-up, AGENTS.md initialization, ecosystem context-file architecture, doc organization, planning sync, and cross-session handoff. Built for [pi](https://github.com/earendil-works/pi-coding-agent).

## Skills

| Skill | Purpose |
|---|---|
| `workspace-catch-up` | Show repo mission, tech stack, current state, last 5 commits, analyze uncommitted files |
| `workspace-ecosystem-claude` | Create ecosystem-wide context-file architecture (AGENTS.md | CLAUDE.md) for multi-repo orgs |
| `workspace-init-agents` | Create or improve a project AGENTS.md with intelligent codebase analysis (host-neutral) |
| `workspace-organize-docs` | Intelligently analyze and organize all .md files in a repository (read-only advisor) |
| `workspace-planning-sync` | Analyze repo and provide intelligent planning documentation suggestions (read-only) |
| `workspace-session-handoff` | Self-contained starter prompt so the next session continues without context loss |

## Install

```bash
pi install npm:@getpipher/workspace
```

Requires [pi](https://github.com/earendil-works/pi-coding-agent).
